Instruction for Using the Architectural Project Manager Platform

The Architectural Project Manager platform is designed for managing projects within an architectural company. Only registered and logged-in users can access the site. Any employee with an active account can create new accounts for other workers.

The Home Page provides an overview of the system with information on the total number of workers, projects, and teams. These numbers are clickable links that lead to dedicated pages with detailed lists.

Navigation Bar

The platform features a top navigation bar with several key buttons:

  • Catalog Button: Displays a dropdown menu containing links to lists of various system objects, such as worker positions, project types, workers, teams, and projects. Each list item is a link to a detailed information page for the corresponding object.
  • Actions Button: Provides access to pages for creating new records, such as adding a new project, registering a worker, or defining a team.
  • Logout Button: Allows users to log out of their account and redirects them to a logout page, which includes a link for re-login if needed.
  • Home Page Button: Redirects users back to the platform's main page.

Detailed Information Pages

Each detailed information page includes all relevant data about a specific object, such as a worker, project, or team. These pages also feature buttons for updating or deleting the corresponding records. For example, a worker's detail page may have options to edit their profile or remove their record from the system.
